Step by step instructions to assemble a home rack server

1. Build a home server rack enclosure to mount your HP or Dell rack server

In this video I will be assembling the iStarUSA 4U open frame rack. This home server rack is only 4u because I will only be including a 1u HP Proliant server to keep power consumption low. Learn how to put together this small server rack or any open frame server rack from 4u to 48u.

2. Rack mount all of the home server components. Home lab essentials assembly

Now that the server rack is built we can start to add in all the Home lab essentials. Lets rack mount the server components into the server cabinet. Attach the server rack kit and install the home network components.

3. Lets Install the Power Supplies into the HP rack server.

Now we install the server box power supplies inside of the server. The goal in this video series is to add the dual server power supply into the Hp Proliant server. This is needed for the server to get power from the electrical outlet.

4. Install Western Digital hard drives into an HP proliant rack server

Lets get our server storage set up. We are using Western Digital 3.5 4tb drives with our g8 server. In this web video we are placing the hard drives into the hard drive hp proliant g8 trays and putting the hard drives with trays into the Hp Proliant g8 server.

5. Power On the Hp Proliant DL360 Server. Time to review the bios setup options.

Its time to learn how to enter the bios. We need to plug in our monitor and keyboard and then were ready to turn on the HP Proliant g8 server and review the bios menus. We are starting to build the computer from the ground up.

6. Build the raids and arrays with the hp array configuration utility

Use f10 HP Provisioning to partition your drives. In this video series we will be partitioning are western digital hard drives into different raid configurations. The Goal is to back up the data and keep the most amount of storage available.

7. Linux Ubuntu fresh install - Clean install ubuntu from usb

Now that we have a hardware raid configurations working correctly we can start to instal an operating system. We will be installing Linux Ubuntu as the OS on the Hp Proliant rack server.
