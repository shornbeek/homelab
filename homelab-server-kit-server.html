Power on the Server

In this guide we turning on the server and taking a quick overview of the bios screens.

Connecting lcd screen with hdmi cable to hp proliant dl360p g8 server

In this video we are looking at the screen and keyboard. These parts are needed to set up the server. We will connect the screen with the hdmi cable to the server using hdmi ports.

Connecting the wireless ubs logitech keyboard to the hp proliant dl360p g8 server

In this video we are connecting the keyboard to the server. Use the Usb ports on the front of the server to plug in your mouse and keyboard. Do this when the server is off.

Power on the hp proliant dl360p g8 server f9 setup bios menu

In this video we are turning on the server! Once the server is turned on you will be able to press the f9 key to see the setup bios.

f9 setup bios menu overview

In this video we are walking through the f9 setup bios. use the arrow keys to move through the utilities. Press enter to select an option. This is a quick overview of the features in the f9 setup bios.
