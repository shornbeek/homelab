Power on a Homelab Server for the first time.

In this guide we are turning on the homelab server for the first time and learning about the servers bios screen utilities.

Step 1. Connecting lcd screen with vga cable to hp proliant dl360p g8 homelab server

In this video we are looking at the lcd screen and usb keyboard. You can hardwire your keyboard or use a wireless usb option. Any Keyboard and mouse comboe should work just fine. You will only need the LCD screen and Keyboard to setup the rack server. After the initial setup you will not need to use the LCD screen and Keyboard. The goal of this homelab server is to access it remotely. Connect the LCD screen with the vga cable to the home lab server using the vga port.

Step 2. Connect the wireless ubs logitech keyboard and mouse to the hp proliant dl360p g8 server

In this video we are connecting the keyboard to the server. This is a really simple set to setting up your homelab. Plug in the USB keyboard dongle to the front usb port on the server. The wireless keyboard will was a good choice for me becasue I an able to move about the room and still interact with the server and LCD screen. Use the Usb ports on the front of the server to plug in your mouse and keyboard. Do this when the server is off. Any mouse and keyboard combo will work just fine. If the mouse and keyboard are both hardwired by usb. Plug in both usb ports into the front of the HP Proliant server.

Step 3. Power on the hp proliant dl360p g8 server to learn about the server bios menu

In this video we are turning on the Hp Proliant Homelab server! We have worked for many videos to assemble all the components of our home network. Now we are ready to press the power button on the frunt of the server to turn on the machine and start to learn about all the different server bios menus. The Server bios menu is like the OS for the server. Its a simple gui that allows you to configure the server hardware before you start to install all of the software. After the server is turned on you will be able to press the f9 key to see the setup home lab server bios gui.

Step 4. Learn about the f9 setup bios menu on an HP Proliant homelab server

In this video we are walking through the f9 setup bios gui. You can use the arrow keys to move through the utilities. Each bios utlitiy has a different function and can be used to configure your home lab server. When you find a utlitiy that you would like to review press enter to select the bios utlity option. This is a quick overview of an HP proliant DL360 g8 bios menu and how to use the f9 setup bios.
