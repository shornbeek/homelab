What is Kit HomeLab ?

At Kit HomeLab we loved the cloud so much that we wanted to build it!

I was really overwhelmed by the amount of server options and parts. I looked for a server kit or a home lab bundle that I could purchase. I could not find an all in one network that meet my needs so I built Kit_1.0.

Kit is a powerful home network that also has room to grow.

The issue with servers and home networks is that people either get really overpowered equipment or they purchase a hot deal that is quickly maxed out or outdated.

I had no desire to build out a large 24u or 48u rack. But I did want to understand rack servers, homelabs, and how the cloud really works. Kit homelab is a mini version of a data center. Google, AWS, and Azure cloud is built on rack servers and blade servers. The blade enclosure is costly so I focused on how to make the best rack server possible.

Kit puts the power of cloud servers like aws right into your hands giving you more control over your data. Kit provides powerful home computing only seen in large data centers.

The use of the 1u HP Proliant DL360 server allows Kit to be thin and powerful. The 48 port 10 GB Unifi fully managed Gigabit switch supports up to 70 Gbps. Kit also has 2u of open space that can be customized to hold additional equipment like more servers, switches, or firewalls.

Kit was invented out of the need to learn about server configurations. Kit is a great foundation to learn how powerful enterprise systems work.

If your into Servers, and Home Networks but still need to stick to a tight budget Kit HomeLab is a great solution. Build a powerful cloud right in your home.
