Rack mount the home server components. Home lab essentials assembly

In this guide we are intalling a Homelab server into a 4u rack enclosure.

Step 1. Start Here DIY server build. Rack all your hardware.

We are have already built the 4u open frame rack. Now we are adding in the components. Lets get started. In this video I discuss what your going to need to assemble the homelab server. I start to attach the rack rail shelf to the 4u open frame rack in the next server video.

Step 2. Installing the NavePoint 1u rack shelf to a 4u rack enclosure

In this video we are adjusting the screws on the NavePoint 1u rack shelf. The goal is to get the shelf lenght correct so that the shelf rails can be attached to the 4u server enclosure. Now that the lenght of the shelf rails are set we can add the NavePoint 1u shelf to the 4u rack enclosure.

Step 3. Attaching the Right NavePoint 1u rack shelf to the rack enclosure

In this video we are adding the rack mount shelf rails to the open frame rack. Start with the rack mount shelf which we will add to our server rack enclosure. We will install the left shelf in the next video. You can install the 1u rack mount shelf on the right side the same way. Dont over tighten the screws to the rack mount. You might need to adjust the shelf location once the server is mounted to the server rack.

Step 4. Attach the left adjustable 1u NavePoint Shelf rack to the server enclosure rails.

In this video we are taking out the right 1u rail shelf to the 4u rack enclosure. The goal is to loosly attach the shelf to the rack wht m6 mat black screws. Learn how to install a rail shelf into a server rack.

Step 5. Use the M6 shelf screws to tighten the 1u Navepoint shelf to the open frame rack enclosure

In this video we are tightening up the screws on the NavePoint Shelf. The goal is to get the m6 shelf screws as tight as possible to hold the server and the 48 port unifi switch inplace. (Note: The rail shelf does take up 1-2mm of space and prevents using the 1u under the shelf rail.)

Step 6. Installing the robot 450 wire clips into the rack enclosure. Cable management clips

The clips will hold the ethernet cables in place. In this video we are talking about adding in our first components the cable managment clips to help control the ethernet cables. You can find the robot 450 wire clips on amazon. If you would like to save $20 you don't have to install the clips. You also have alot of robot 450 wire clips left over when your done with the clip install.

Step 7. Cyberpower Power Supply and 7 inch rack Shelf review. Unplug the power supply/

Lets take a look at the CyberPower RKBS15S2F10R 15A 12-Outlet 1U RM Rackbar Surge Supressor. This is a great well build poewr supply that we will be reviewing. Make sure your CyberPower Surge Supressor is unplugged before you start the install. We will also take a quick look at the 7inch 1u rack shelf made by StarTech.com. The 1U Server Rack Mount Shelf - 7in Deep Fixed Steel Universal Tray for 19" AV, Data & Network Equipment.

Step 8. Install a 1U Server Rack Mount Shelf that is 7in Deep to the rack server enclosure.

In this video we are adding in the 1u 7 inch rack mount shelf. The 1u shelf will be added to the bottom 1u of the rack enclosure. Install a network rack shelf with m5 or m6 screws. the 7 inch rack shelf depth was chosen because any thing deaper touch the HP Proliant server.

Step 9. Installing the CyberPower 12-Outlet 1U Rackbar Surge Supressor into a 4u rack enclosure

In this video we are adding in the cyber power supressor into the 4u rack enclosure. We are using the black m6 screws. The first thing you have to do is install the rack ears onto the CyberPower Surge Supressor. Screw on the 2 rack ears to the power supply. Attach the rack ears on the CyperPower Power supply to the open frame rack enclosure. Screw 4 m6 screws and washers on tight.

Step 10. Add a Modem to the server with ethernet cables inside the rack enclosure.

In this video we are adding the modem to the Homelab. Most modums are provided by your internet company. This video shows the modems location on the rack shelf in the homelab. It also teaches you how to install the cat 6 ethernet cables into the modem and how you plug in the modem power supply. Okay graet your modem is now added to your rack enclosure.

Step 11. Connect the modem to the router with a cat 6 ethernet cable

In this video we are connecting the router to the modem using a ethernet cable. We need to give the router a hard wired connection to the modem. This is done through an ultra thin cat6 cable. You can run the thin cable through the cable clips installed on the inside of the rack enclosure.

Step 12. Install Unifi 48 Port Unifi PoE Switch into a server rack enclosure.

In this video we are mounting our switch by placing it onto the shelf. The shelf bolts had to be readjusted in order to fit the switch. Cat 6 cables were added to the switch. These 6 ethernet cables from the switch will be attached to the back of the HP Proliant server.

Step 13. Hp Proliant DL360 G8 Install to 4u rack

In this video we are installing the Dl360 G8 onto the 4u rack above the 48 port switch. Be sure to add in the crossbar first before you slide in the server.

Step 14. Hooking up the DL360 G8 to the Unifi 48 port switch

In this video we are attaching the DL360 server to the switch and power supply.

Step 15. Kit_1.0 hardware completed

In this video we have finalized all of the compenents to install on the 4u rack. You can add on the 1u blank panels for show if you would like.
