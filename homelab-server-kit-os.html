From USB boot to install linux ubuntu operating system on server

This server tutorial teaches how to install an operating system for a usb drive onto your home lab server.

Step 1. Download linux ubuntu from ubuntu.com and save the ubuntu iso file to your computer

In this video we are using chrome to go to ubuntu.com to download our linux operating system. Download the most recent desktop version to your computer. Once the ubuntu files are on your computer your ready to move on to the next step. Download Ubuntu Desktop Download the latest LTS version of Ubuntu, for desktop PCs and laptops. LTS stands for long-term support until April 2023, of free security and maintenance updates, guaranteed.

Step 2. Download etcher to flash linux ubuntu iso file to usb drive for server install

In this video we are using chrome to go to Etcher Website to download the software. Once the etcher program is installed on your computer you then load the linux ubuntu iso file through etcher. Be sure to have a flash drive plugged into your computer. The etcher program will erase anything on the USB drive so have a new drive ready or have all of your data backed up. The drive I used was 8gb.

Step 3. Insert flashed USB into hp proliant server to install linux ubuntu operating system.

In this video are removing our flashed drive where we justed used etcher to install the linux ubuntu iso file. Now remove the drive from your computer (mac book pro) and placing the flashed etcher usb into one the the USB drives HP proliant dl360p g8 server. Now that the USB drive is installed we are able to move onto the next step and power on the Hp Proliant g8 homelab.

Step 4. USB boot to install linux ubuntu operating system on an HP Proliant rack server. Part 1

In this video are turning on the server and booting from the jump drive. The goal is to have the server find the usb drive on its own. Don't press anything and let the server find do the work for you. Once the Ubuntu desktop screen is displayed your ready to move to the next video.

Step 5. Install linux ubuntu software onto hp proliant dl360p g8 rack server. Part 2

In this video are walking through the steps on how to install linux ubuntu onto the hp server. Be sure to reboot the server once or twice to make sure ubuntu was installed correctly. Be sure to remember you user name and password. You will need it as we continue to build this hp proliant dl360.
